Definições de Conceitos no Design

Affordance

O conceito de affordance refere-se a como as características físicas de um objeto sugerem sua função. Por exemplo, um botão é projetado para ser pressionado.

Semiótica

A semiótica estuda os signos e símbolos. Um ícone de "lixeira" é um exemplo claro, pois comunica a ação de descartar algo.

Usabilidade

Usabilidade é a facilidade com que uma interface pode ser utilizada. Um site com boa usabilidade é intuitivo e fácil de navegar.

CSS

CSS é uma linguagem de estilo usada para definir a aparência de um documento HTML. Ele controla cores, fontes, espaçamentos e layouts.

CSS Inline – Aplicado diretamente na tag HTML usando o atributo style. Exemplo:
Útil para estilos rápidos. Dificulta a manutenção do código.

CSS Interno (Embebido) – Definido dentro da tag style no head do HTML.
Funciona bem para páginas simples. Pode tornar o HTML poluído se muito usado.

CSS Externo – Arquivo .css separado, vinculado ao HTML via link
Melhor prática, facilita a manutenção e reutilização do código. Requer um arquivo externo.

A escolha depende da necessidade, mas o CSS externo é o mais recomendado para projetos maiores.
